# 花语时光功能说明文档

## 1. 产品定位

花语时光系统用于鲜花店品牌展示、作品展示、门店信息展示、关于我们内容运营，以及后台内容维护和访客留言收集。

系统不包含电商交易能力，不支持：

- 购物车
- 下单支付
- 库存管理
- 会员中心
- 订单履约

## 2. 终端范围

系统包含以下终端：

- PC Web 前台
- PC Web 管理后台
- 微信小程序前台

## 3. Web 前台功能

### 3.1 首页

首页主要包含：

- 品牌首屏展示
- 背景图与主视觉
- 品牌文案
- 统计数据展示
- 精选作品展示
- 服务场景分类入口
- 品牌故事概览
- 门店信息入口

主要用途：

- 快速传达品牌调性
- 引导进入作品画廊
- 呈现门店和服务能力

### 3.2 作品画廊

作品画廊支持：

- 分类筛选
- 关键词搜索
- 排序切换
- 分页浏览
- 进入作品详情页

排序能力包括：

- 精选优先
- 最新
- 价格升序
- 价格降序

### 3.3 作品详情

作品详情页展示：

- 多图画廊
- 作品名称
- 分类
- 价格
- 花材信息
- 寓意说明
- 标签
- 相关推荐

### 3.4 关于我们

关于页展示：

- 页首主视觉
- 标题与副标题
- 品牌故事正文
- 时间轴
- 团队成员
- 门店相关信息

该页内容已经支持后台动态维护。

### 3.5 联系我们

联系页包含：

- 门店地址
- 电话
- 微信
- 地图位置
- 营业时间
- 留言表单

留言表单提交后会进入后台“用户留言”模块。

## 4. Web 管理后台功能

### 4.1 管理员登录

管理员可通过 `/admin/login` 登录后台。

登录成功后：

- 浏览器保存 Bearer Token
- 后续后台请求自动携带认证头

### 4.2 运营总览

后台首页为运营总览，主要用于查看：

- 作品总数
- 精选作品占比
- 分类分布
- 最新作品
- 未读留言数量
- 首页主视觉和品牌信息概览
- 快捷入口

### 4.3 作品管理

作品管理支持：

- 作品列表查看
- 搜索与筛选
- 新增作品
- 编辑作品
- 删除作品
- 上传作品图片
- 维护精选状态
- 调整排序

作品字段包括：

- 编号
- 名称
- 分类
- 图片列表
- 价格
- 描述
- 花材
- 寓意
- 标签
- 是否精选
- 排序
- 创建时间

### 4.4 内容配置

内容配置是当前后台的核心内容管理入口，统一维护以下内容：

#### 首页内容

- 品牌名称
- 首页小标语
- 首页主标题
- 首页简介
- 首页主视觉图
- 按钮文案
- 统计数据
- 联系简介
- 营业时间摘要
- 页脚简介

#### 门店信息

- 电话
- 微信
- 地址
- 纬度
- 经度
- 营业时间

#### 品牌故事

- 标题
- 副标题
- 正文
- 配图

#### 关于我们页

- 页首图
- 页首小标语
- 页首标题
- 页首副标题
- 品牌故事标题
- 品牌故事正文
- 时间轴
- 团队成员

### 4.5 用户留言

后台可查看用户从联系页提交的留言。

支持能力：

- 列表查看
- 关键字搜索
- 已读 / 未读筛选
- 将留言标记为已读
- 查看最近留言

留言字段包括：

- 留言编号
- 姓名
- 电话
- 留言内容
- 提交时间
- 已读时间

### 5.3 作品详情

- 多图画廊
- 花材说明
- 寓意说明
- 相关推荐

### 5.4 关于我们

- 品牌故事
- 门店信息

## 6. 系统运营流程

### 6.1 内容运营流程

典型运营流程如下：

1. 管理员登录后台
2. 在“作品管理”中维护作品
3. 在“内容配置”中更新首页、门店和关于我们
4. 访客在前台浏览作品或提交留言
5. 管理员在“用户留言”中处理咨询与预约

### 6.2 数据流向

- 前台浏览内容读取公开 API
- 后台编辑内容调用受保护 API
- 更新结果写入 MySQL
- 上传文件保存在 uploads 目录

## 7. 非功能约束

### 7.1 当前适用场景

当前系统适合：

- 品牌展示站
- 门店作品展示
- 内容运营后台
- 小规模线索收集

### 7.2 当前不覆盖能力

当前版本不适合：

- 电商交易
- 高并发订单系统
- 多角色复杂权限体系
- ERP/库存/供应链场景
