# 花语时光功能说明文档

## 1. 产品定位

花语时光系统用于鲜花店品牌展示、作品展示、门店信息展示、关于我们内容运营，以及后台内容维护和访客留言收集。

系统不包含电商交易能力，不支持：

- 购物车
- 下单支付
- 库存管理
- 会员中心
- 订单履约

## 2. 终端范围

系统包含以下终端：

- PC Web 前台
- PC Web 管理后台
- 微信小程序前台

## 3. Web 前台功能

### 3.1 首页

首页主要包含：

- 品牌首屏展示
- 背景图与主视觉
- 品牌文案
- 统计数据展示
- 精选作品展示
- 服务场景分类入口
- 品牌故事概览
- 门店信息入口

主要用途：

- 快速传达品牌调性
- 引导进入作品画廊
- 呈现门店和服务能力

补充说明：

- 首页统计数据不由后台人工填写
- 前台会按分页聚合全部作品后计算真实分类数、精选数与作品总量

### 3.2 作品画廊

作品画廊支持：

- 分类筛选
- 关键词搜索
- 排序切换
- 分页浏览
- 进入作品详情页

排序能力包括：

- 精选优先
- 最新
- 价格升序
- 价格降序

### 3.3 作品详情

作品详情页展示：

- 多图画廊
- 作品名称
- 分类
- 价格
- 花材信息
- 寓意说明
- 标签
- 相关推荐

说明：

- 详情页不再对访客直接展示内部作品编号
- 页面保留参考说明，用于引导用户理解价格与花材会随季节和制作需求调整

### 3.4 关于我们

关于页展示：

- 页首主视觉
- 标题与副标题
- 品牌故事正文
- 时间轴
- 团队成员
- 门店相关信息

该页内容已经支持后台动态维护。

### 3.5 联系我们

联系页包含：

- 门店地址
- 电话
- 微信
- 地图位置
- 营业时间
- 留言表单

留言表单提交后会进入后台“用户留言”模块。

## 4. Web 管理后台功能

### 4.1 管理员登录

管理员可通过 `/admin/login` 登录后台。

登录成功后：

- 浏览器保存 Bearer Token
- 后续后台请求自动携带认证头

### 4.2 运营总览

后台首页为运营总览，主要用于查看：

- 作品总数
- 精选作品占比
- 分类分布
- 最新作品
- 未读留言数量
- 首页主视觉和品牌信息概览
- 快捷入口

### 4.3 作品管理

作品管理支持：

- 作品列表查看
- 搜索与筛选
- 新增作品
- 编辑作品
- 删除作品
- 上传作品图片
- 维护精选状态
- 调整排序

作品字段包括：

- 编号
- 名称
- 分类
- 图片列表
- 价格
- 描述
- 花材
- 寓意
- 标签
- 是否精选
- 排序
- 创建时间

### 4.4 站点配置

站点配置是当前后台的核心内容管理入口，统一维护以下内容：

#### 首页内容

- 品牌名称
- 首页小标语
- 首页主标题
- 首页简介
- 首页主视觉图
- 按钮文案
- 联系简介
- 营业时间摘要
- 页脚简介

#### 门店信息

- 电话
- 微信
- 地址
- 纬度
- 经度
- 营业时间

#### 品牌故事

- 标题
- 副标题
- 正文
- 配图

#### 关于我们页

- 页首图
- 页首小标语
- 页首标题
- 页首副标题
- 品牌故事标题
- 品牌故事正文
- 时间轴
- 团队成员

补充：

- 首页统计数据已取消后台人工配置
- 真实统计由系统根据全部作品数据自动计算

### 4.5 AI 生图配置

后台独立提供“AI 生图配置”一级入口，用于维护：

- AI 开关
- 提供商
- API Key
- 生图模型
- 文生图/以图生图接口参数
- 作品信息建议模型参数

### 4.6 系统状态与保护面板

后台“系统状态”页面当前支持查看：

- 当前版本、部署环境、提交号、部署时间
- 数据库状态
- 上传目录状态
- AI 配置状态
- 最近备份状态
- 操作日志保留与归档状态
- 并发保护状态

其中并发保护状态面板会展示：

- 是否启用基础保护
- 公开读取 / 公开写入 / 后台 / 高成本接口限流阈值
- AI / 上传 / 配置导入并发阈值
- 已触发限流次数
- 已触发繁忙拒绝次数

该页面用于部署后巡检、容量观察和售后排障。

### 4.7 基础稳定性保护

当前系统已经内置以下稳定性能力：

- 全站路由分级限流
- 高成本接口并发隔离
- 公开只读接口本地缓存
- 后台系统状态页保护统计

主要目标：

- 避免短时间大量请求直接压垮单机服务
- 降低 AI、生图上传、配置导入等重接口堆积风险
- 为后续多实例、Redis、网关限流扩展保留边界

### 4.6 用户留言

后台可查看用户从联系页提交的留言。

支持能力：

- 列表查看
- 关键字搜索
- 已读 / 未读筛选
- 将留言标记为已读
- 查看最近留言

留言字段包括：

- 记录标识
- 姓名
- 电话
- 留言内容
- 提交时间
- 已读时间

### 4.7 系统状态

后台系统状态页支持查看：

- 当前部署版本
- 数据库连接、版本与容量
- 上传目录状态、文件数与容量
- 磁盘总量、可用量与已用比例
- AI 配置启用状态
- 最近备份目录、更新时间与下载入口
- 自动轮询状态与最近错误

### 4.8 操作日志

后台操作日志页支持：

- 后台写操作与登录行为筛选
- 查看日志详情、请求摘要、前后快照与字段差异
- 导出 CSV
- 对支持的结构化数据按单条日志恢复
- 记录恢复链路，便于后续追踪

### 5.3 作品详情

### 5.4 关于我们

## 6. 系统运营流程

### 6.1 内容运营流程

典型运营流程如下：

1. 管理员登录后台
2. 在“作品管理”中维护作品
3. 在“站点配置”中更新首页、门店和关于我们
4. 访客在前台浏览作品或提交留言
5. 管理员在“用户留言”中处理咨询与预约

### 6.2 数据流向

- 前台浏览内容读取公开 API
- 后台编辑内容调用受保护 API
- 更新结果写入 MySQL
- 上传文件保存在 uploads 目录

## 7. 非功能约束

### 7.1 当前适用场景

当前系统适合：

- 品牌展示站
- 门店作品展示
- 内容运营后台
- 小规模线索收集

### 7.2 当前不覆盖能力

当前版本不适合：

- 电商交易
- 高并发订单系统
- 多角色复杂权限体系
- ERP/库存/供应链场景
