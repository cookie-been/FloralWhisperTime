# 素材交付规范

本文档用于规范项目交付时的图片、Logo 和展示素材准备方式，便于客户、设计和实施人员协同。

## 1. 适用范围

当前系统中需要重点准备的视觉素材包括：

- 品牌 Logo
- 首页主图
- 首页轮播图
- 管理后台登录页轮播图
- 联系页展示图
- 品牌故事配图
- About 页首图
- 作品图片
- 团队成员头像

## 2. 通用要求

所有素材建议满足：

- 清晰
- 风格统一
- 不带无关水印
- 不带明显压缩噪点
- 不使用过暗、过糊、过度裁切的图片

建议格式：

- `jpg`
- `jpeg`
- `png`
- `webp`

上传限制：

- 单张文件不超过 `20MB`

## 3. Logo 规范

用途：

- 前台导航 Logo
- 后台左上角 Logo
- 浏览器标签图标来源

建议：

- 比例尽量接近 `1:1`
- 建议最小尺寸不低于 `512 x 512`
- 主题主体保持居中
- 四周留出安全边距

命名建议：

```text
brand-logo.png
```

## 4. 首页主图规范

用途：

- 首页首屏单图兜底

建议：

- 画面主题直接体现品牌或花艺场景
- 建议横向图
- 建议尺寸不低于 `1920 x 1080`
- 左侧或中部应预留文字可读区域

## 5. 首页轮播图规范

用途：

- 首页首屏轮播展示

建议数量：

- `3` 到 `4` 张最佳

建议：

- 尺寸比例尽量统一
- 建议横向大图
- 建议尺寸不低于 `1920 x 1080`
- 色调保持同一品牌风格
- 图片上不要堆太多自带文字

内容建议：

- 门店空间
- 花束场景
- 婚礼或活动现场
- 花艺制作细节

命名建议：

```text
home-hero-01.jpg
home-hero-02.jpg
home-hero-03.jpg
```

## 6. 后台登录页轮播图规范

用途：

- 管理后台登录页背景轮播

建议数量：

- `2` 到 `3` 张

建议：

- 风格比前台首页更克制、更稳重
- 与品牌风格一致
- 避免太花或太杂乱
- 避免主体过满导致登录框难以突出

命名建议：

```text
admin-login-01.jpg
admin-login-02.jpg
```

## 7. 联系页展示图规范

用途：

- 联系我们页右侧或辅助展示区

建议数量：

- `2` 到 `4` 张

建议：

- 展示门店环境、包装细节、花束近景
- 适合纵向或近景构图
- 统一色调

命名建议：

```text
contact-01.jpg
contact-02.jpg
contact-03.jpg
```

## 8. 品牌故事配图规范

用途：

- 首页品牌故事区
- 品牌故事相关内容区

建议数量：

- `2` 到 `4` 张

建议：

- 可展示主理人、门店、创作过程、代表作品
- 画面要有品牌感，不要只放素材图库式空镜

命名建议：

```text
story-01.jpg
story-02.jpg
story-03.jpg
```

## 9. About 页首图规范

用途：

- 关于我们页第一屏背景图

建议：

- 要能直接表达品牌气质
- 建议横向大图
- 预留标题与副标题阅读空间
- 不要用过度花哨的拼贴图

命名建议：

```text
about-hero.jpg
```

## 10. 作品图片规范

用途：

- 作品列表封面
- 作品详情页画廊

建议：

- 每个作品至少 `1` 张，建议 `3` 到 `5` 张
- 首图优先作为封面图
- 保持主体明确
- 尽量不要混入太多无关背景

建议尺寸：

- 单张建议不低于 `1200px` 宽

命名建议：

```text
wedding-001-cover.jpg
wedding-001-detail-01.jpg
wedding-001-detail-02.jpg
```

## 11. 团队头像规范

用途：

- About 团队成员展示

建议：

- 比例接近 `1:1`
- 人物面部清晰
- 风格统一
- 背景尽量简洁

命名建议：

```text
team-alice.jpg
team-bob.jpg
```

## 12. 交付建议

建议客户交付素材时按目录整理：

```text
assets/
  logo/
  home/
  admin-login/
  contact/
  story/
  about/
  flowers/
  team/
```

## 13. 常见问题

### 13.1 图片太大

建议先压缩后再上传，避免浪费上传时间和存储空间。

### 13.2 尺寸不统一

同一组轮播图、联系页图、故事图区的图片，尽量统一比例和风格，否则前台观感会不整齐。

### 13.3 图片自带很多字

不建议，系统前台本身已经有文案层，图片再带大段字会影响展示效果。
