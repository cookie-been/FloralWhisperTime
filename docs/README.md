如果你只想记最少内容，直接记这 4 条：

1. 命令入口只记 `./ops.sh`
2. 后台低风险动作看“运维中心”
3. 部署、升级、回滚仍走 Linux / Docker 命令行
4. 文档入口优先看 [从这里开始](./start-here.md)
