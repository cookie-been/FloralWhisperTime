## 10. 风险与回滚建议

### 已知风险点

- 如果环境变量仍保留默认值，正式环境存在安全风险
- 如果 AI Key、模型和接口地址不匹配，AI 相关能力不可用
- 如果上传目录权限错误，图片上传与 AI 图片落盘会失败

### 回滚建议

- 源码模式先执行：

```bash
./rollback.sh --latest --dry-run
```

- release 模式使用：

```bash
./release-rollback.sh --latest-previous
```
